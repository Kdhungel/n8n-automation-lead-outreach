# n8n Automation Lead Outreach

This repository contains an n8n workflow for automated lead generation, auditing websites, and sending personalized outreach emails.

## Project Structure
- workflow/: contains the exported n8n workflow JSON
- docs/: project documentation
- assets/: screenshots, visuals

## Workflow Highlights
- Lead generation
- Website auditing
- Personalized email creation
- Priority filtering (Hot / HighScore leads)
- Summary table output
- Optional Gmail integration for sending emails
- Logging of sent emails

## How to Use
1. Open n8n.
2. Import `workflow/lead-outreach-workflow.json`.
3. Inspect and run workflow nodes.
4. Optionally connect Gmail for real email sending.
