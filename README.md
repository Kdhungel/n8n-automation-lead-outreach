# n8n Automation Lead Outreach

This repository contains an n8n workflow project for automated lead generation, website audits, and personalized email outreach.

For detailed documentation, see the [docs/README.md](docs/README.md).

## Quick Setup
1. Clone the repo:
   ```bash
   git clone https://github.com/Kdhungel/n8n-automation-lead-outreach.git
   ```
2. Open n8n and import the workflow from workflow/
3. Review workflow nodes and run for testing
